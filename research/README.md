# Gravel Race Database Research Repository

## Overview

This repository contains research data for the 388-race gravel cycling database. Research is organized by priority, region, and research status.

## Research Structure

```
research/
├── README.md (this file)
├── top_priority/          # Tier 1 and zero-competition races
├── by_region/            # Research organized by geographic region
├── by_protocol/          # Research organized by protocol fit
├── uci_series/          # UCI Gravel World Series research
├── findings/             # Research findings and updates
└── scripts/              # Research automation scripts
```

## Research Fields

For each race, research:
- [ ] Official website URL
- [ ] Registration URL
- [ ] Confirmed 2026 date
- [ ] Field size (actual numbers)
- [ ] Entry cost (current pricing)
- [ ] RideWithGPS route ID
- [ ] Course description
- [ ] Elevation profile
- [ ] Prize purse (if applicable)
- [ ] Series affiliation
- [ ] UCI qualifier status

## Research Status

- **Pending**: Not yet researched
- **In Progress**: Research started
- **Complete**: All fields researched
- **Verified**: Data confirmed from official sources
