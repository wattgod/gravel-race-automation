Build Phase Begins

Hey {athlete_name},

You're entering the build phase. This is where the base fitness you've built gets sharpened into race-ready form.

What to Expect

  • Higher intensity sessions. VO2max intervals, threshold work, and sweet spot efforts.
  • Longer weekend rides. These should include race nutrition practice.
  • More fatigue. This is normal. Recovery weeks are built in.

Key Adjustments

  • Fueling: Increase carb intake on hard days. You can't build fitness on empty.
  • Sleep: Prioritize 8+ hours. This is non-negotiable during build phase.
  • Listen to your body: If you're too fatigued to hit power targets, take an extra easy day. Missing one session is better than overtraining for a week.

The work you put in now shows up on race day. Let's go.
