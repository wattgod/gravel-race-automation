Gravel God Embed Widget

Add a Gravel God race rating badge to your site. Compact, lightweight, and self-contained.

Quick Start

Paste this where you want the badge to appear:

<div class="gg-embed" data-slug="unbound-200"></div>
<script src="https://gravelgodcycling.com/embed/gg-embed.js" async></script>

Change data-slug to any race slug from the race database.

Multiple Badges

Include the script once, add as many badges as you want:

<div class="gg-embed" data-slug="unbound-200"></div>
<div class="gg-embed" data-slug="leadville-100"></div>
<div class="gg-embed" data-slug="crusher-in-the-tushar"></div>
<script src="https://gravelgodcycling.com/embed/gg-embed.js" async></script>

How It Works

  • The script loads a compact JSON file (~15KB) with all 328 race ratings
  • Each badge is rendered client-side with race name, tier, score, location, and date
  • Clicking the badge links to the full race profile on Gravel God
  • No dependencies, no cookies, no tracking (GA4 event only on gravelgodcycling.com)
  • Styles are scoped to .gg-embed-card and won't affect your site

Available Slugs

Find any race slug at https://gravelgodcycling.com/gravel-races/. The slug is the last part of the race profile URL.
